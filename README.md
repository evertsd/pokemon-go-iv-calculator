### Getting started
```
  npm install
```

## Terminal Usage

### Parse single result
```
  node index.js zubat 342 52 3000
```

### Parse CSV file
- Follow this format
```
name,cp,hp,stardust,level
raticate,815,74,2500
```
- Run this command
```
  node index.js /path/to/input.csv > /path/to/output.csv
```
